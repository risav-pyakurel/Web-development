Selectors in html

Element selector

It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them based on the element name (or tag) for example p, h1, div, span, etc.0

ID selector

CSS id selectors select any element in an HTML page, if it has an attribute called id, whose value matches the name of the id selector. In a stylesheet document, the name of the id selector is preceded by a "#". If an id selector is combined with another or more selectors, id selectors must be preceded with a #. IDs can only be applied once in an HTML page

Class selector

The CSS class selector matches elements based on the contents of their class attribute.
